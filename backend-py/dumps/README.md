# Dumps Directory

This directory stores JSON dumps of scraped data.

## Files

- `_dump_checklist.json` - Master checklist tracking all dumps
- `{type}_{timestamp}.json` - Individual dump files

## Managed By

JSONDumpManager automatically creates, tracks, and manages files here.

## Backup

Consider backing up important dumps to external storage.

## Cleanup

Use the "Clear All" button in the UI or API to delete old dumps.
